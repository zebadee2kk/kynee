# Architecture Decision Records (ADRs)

## Purpose

This directory contains Architecture Decision Records documenting significant technical decisions made during KYNEÄ’ development.

## Format

Each ADR follows this template:

```markdown
# ADR-XXXX: Title

## Status
[Proposed | Accepted | Deprecated | Superseded]

## Context
What is the issue we're addressing?

## Decision
What did we decide?

## Consequences
What are the trade-offs?
```
